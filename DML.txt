DML COMMANDS:DATA MANUPULATION LANGUAGE

INSERT,UPDATE,DELETE



1.INSERT:TO INSERT VALUES TO THE TABLES
SYNTAX:
INSERT INTO TABLE_NAME VALUES (VALUE1,VALUE2.....);
EXAMPLE:
SINGLE_ROW:
INSERT INTO STUDENTS VALUES (1,'ANEELA',22,1234567);

FOR MULTIPLE ROWS:
INSERT INTO STUDENTS VALUES (1,'ANEELA',22,1234567),(2,'AKHILA',20,123456),(3,'NARESH',24,123456);



2.UPDATE:TO UPDATE THE VALUES TO THE TABLE


SYNTAX:
UPDATE TABLE_NAME SET COLUMN_NAME=VALUE WHERE CONDITION


EXAMPLE:
UPDATE STUDENTS SET AGE=19 WHERE SNAME='AKHILA';


UPDATE STUDENTS SET SNAME='ANEE' WHERE SNAME='ANEELA';



3.DELETE:DELETE A(SINGLE) ROW FROM THE TABLE

SYNTAX:
DELETE FROM TABLE_NAME WHERE COLMN_NAME;


EXAMPLE:
DELETE FROM STUDENTS WHERE AGE=24;
